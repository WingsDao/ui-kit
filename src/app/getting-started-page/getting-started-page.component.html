Basic Setup

Step 1: Install library


npm install @wingsplaftorm/ui-kit
or
yarn add @wingsplaftorm/ui-kit

Step 2: Import library module

import UiKitModule and FormsModule

Step 3: Configure global-styles

import ws-global-styles component somewhere in a your root component template

Great! Now you are ready to use ui-kit components!
